# Python 의 특징

이 교재에서는 ROS2를 활용해 프로그램을 만들고 로봇을 제어합니다.

ROS2에서 프로그램을 작성하는 대표적인 방법은 크게 두 가지입니다.

- Python
- C++

두 언어 모두 ROS2에서 공식적으로 지원되며 실제 현업에서도 많이 사용됩니다.

하지만 학습 관점에서 보면 Python이 훨씬 접근성이 좋습니다.

문법이 간결하고 읽기 쉬우며, 처음 프로그래밍을 접하는 사람도 비교적 빠르게 익힐 수 있기 때문입니다.

이 교재에서는 ROS2의 구조와 동작 원리를 이해하는 데 집중하기 위해 Python을 기준으로 설명합니다.

#### 파이썬의 장점

Python은 현재 가장 많이 사용되는 프로그래밍 언어 중 하나이며, 다양한 장점을 가지고 있습니다.

- 배우기 쉽고 읽기 쉬운 문법
- 방대한 라이브러리 생태계
- 운영체제에 관계없는 높은 호환성
- 인터프리터 기반의 빠른 개발 속도
- 다양한 프로그래밍 패러다임 지원
- AI와 데이터 과학 분야의 중심 언어
- 거대한 글로벌 커뮤니티
- 다른 언어와의 연동이 쉬움

이러한 특징 때문에 Python은 자동화, 웹 개발, 데이터 분석, 인공지능, 그리고 ROS2까지 매우 넓은 분야에서 활용되고 있습니다.

---

#### 들여쓰기 기반 문법

Python의 가장 큰 특징 중 하나는 중괄호(`{}`) 를 사용하지 않는다는 점입니다.

대신 들여쓰기(Indent)를 이용하여 코드의 범위를 구분합니다.

```python
if speed > 10:
	print("Robot Move")
	print("Speed:", speed)

print("Program End")
```

위 코드에서 들여쓰기 된 부분은 `if` 문에 포함되며, 들여쓰기가 끝나는 순간 `if` 문도 종료됩니다.

일반적으로 Python에서는 공백 4칸을 사용하며, 대부분의 개발 환경에서는 `Tab` 키를 사용하여 자동으로 들여쓰기를 입력합니다.

---

#### 인터프리터 언어

Python은 인터프리터 기반 언어입니다.

C++처럼 컴파일 과정을 거쳐 실행 파일을 만드는 방식이 아니라 작성한 코드를 바로 실행할 수 있습니다.

```python
python3 main.py
```

이러한 특징 때문에 개발 속도가 빠르고 테스트와 실험을 반복하기 쉽습니다.

반면 실행 성능은 일반적으로 컴파일 언어인 C++보다 다소 불리한 편입니다.

---

#### 동적 타입 언어

Python은 변수의 자료형을 미리 선언하지 않습니다.

```python
speed = 100
name = "SO ARM101"
```

C++ 처럼 `int`, `float`, `string` 등을 미리 선언할 필요가 없기 때문에 코드를 간결하게 작성할 수 있습니다.

이러한 특징은 입문자가 프로그래밍을 시작하기 쉽게 만들어 주는 장점 중 하나입니다.

---

#### 모든 것은 객체이다.

Python에서는 숫자, 문자열, 리스트와 같은 기본 자료형조차 객체(Object)로 취급됩니다.

```python
speed = 100
name = "Robot"
```

뿐만 아니라 ROS2에서 사용하게 될 Node, Message, Service 역시 모두 객체 형태로 동작합니다.

```python
node = Node("robot_node")
msg = JointState()
```

따라서 Python에서 객체와 클래스를 이해하는 것은 ROS2를 이해하는 데에도 큰 도움이 됩니다.

#### 왜 Python으로 시작하는가?

Python의 창시자인 귀도 반 로섬(Guido van Rossum)은 이런 말을 남겼습니다.

> “C언어로 코딩하기에는 인생이 너무 짧다.”
> 

이 말은 Python이 얼마나 생산성과 접근성이 높은 언어인지 잘 보여줍니다.

물론 Python은 쉬운 문법과 빠른 개발 속도를 제공하는 대신, 실행 성능에서는 C++보다 불리한 부분이 있습니다.

특히 아래와 같은 경우에는 C++이 더 유리합니다.

- 고속 연산이 필요한 경우
- 실시간성이 중요한 경우
- 하드웨어에 가까운 저수준 제어가 필요한 경우

ROS에서도 이러한 이유로 성능이 중요한 노드나 라이브러리는 C++로 작성되는 경우가 많습니다.

하지만 ROS2를 배우는 초기 단계에서는 성능보다 구조와 개념을 이해하는 것이 훨씬 중요합니다.

따라서 이 교재에서는 Python을 중심으로 학습을 진행하고, 필요에 따라 C++ 내용을 추가로 설명합니다.
