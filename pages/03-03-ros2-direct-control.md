#### Python을 이용한 ROS2 패키지 제작

이번 장에서는 Python을 이용해 ROS2 패키지와 노드를 직접 만들어봅니다. 먼저 Workspace와 Package 구조를 살펴본 뒤 Turtlesim을 활용하여 Publisher와 Subscriber 노드를 작성합니다.

이후 여러 노드를 한 번에 실행하는 Launch 파일과 커스텀 인터페이스를 만들고, Service와 Action의 Server 및 Client 노드를 구현합니다.

- ROS2 Workspace와 Package 생성
- Publisher와 Subscriber 노드 작성
- Publisher와 Subscriber를 결합한 노드 제작
- Launch 파일을 이용한 다중 노드 실행
- 커스텀 메시지, 서비스, 액션 정의
- Service Server와 Client 구현
- Action Server와 Client 구현
